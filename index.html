What is callback ?

In JavaScript, a callback is a function passed into another function as an argument to be executed later.

This example illustrates how callback functions work. Here, in above image, displayerFunction will be called in myCalculator function. So, when the calculation in the myCalculator is over then the result will be sent to displayerFunction as a parameter. Otherwise, it is no use to call displayerFunction since it needs the sum of the calculation to display.
